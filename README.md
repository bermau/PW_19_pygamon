# Yet Another Pygamon

Fortement inspiré d'une vidéo par [Graven](https://www.youtube.com/watch?v=ooITOxbYVTo&t=1317s).

# Views
![The world](./images/world.png)

# Starting the game
1) Download this files.
2) You need Python 3 and some librairies: pygame, pyscroll, pytmx.
3) start `/src/main.py`.
